# Análisis de Datos de Escuelas y Notas

Este repositorio contiene el trabajo práctico de análisis de datos enfocado en el rendimiento de estudiantes en distintas escuelas. Se utilizan técnicas de análisis exploratorio de datos (EDA) y visualización para entender patrones, tendencias y posibles áreas de mejora en la educación.

## Contenido del Proyecto

- `TP análisis de datos.ipynb`: Notebook principal donde se realiza todo el análisis.
- `README.md`: Este archivo con la descripción del proyecto.

## Objetivos

1. Analizar los resultados académicos de los estudiantes por escuela y materia.
2. Identificar tendencias y patrones en los resultados.
3. Visualizar datos de manera clara y comprensible.
4. Extraer conclusiones que puedan ayudar a la toma de decisiones educativas.

## Tecnologías y Herramientas

- Python 3.x
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Estructura del Notebook

1. **Carga de Datos**: Importación de los datasets de notas y escuelas.
2. **Limpieza de Datos**: Manejo de valores faltantes y corrección de errores.
3. **Análisis Exploratorio**: Estadísticas descriptivas, distribución de notas, comparación entre escuelas.
4. **Visualización**: Gráficos de barras, histogramas, boxplots y otras representaciones visuales.
5. **Conclusiones**: Resumen de hallazgos importantes y posibles recomendaciones.
